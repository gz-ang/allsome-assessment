# Order Statistics Generator

This project is an assessment done by Ang Guang Zheng for job application with Allsome Malaysia

## Installation

Clone the project to your local directory

```bash
git clone https://github.com/gz-ang/allsome-assessment.git
```

## Requirements

The project was developed with PHP8.2, but should be compatible with older PHP7.x versions

## Usage
Open a terminal and change to project directory

or

Open the project folder using any IDE and open the integrated terminal

Run the following command

```php
php -f index.php
```
